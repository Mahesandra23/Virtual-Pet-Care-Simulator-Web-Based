# 🐾 Virtual Pet Care Simulator

**Virtual Pet Care Simulator** is a web-based game project inspired by the classic *Tamagotchi* concept. Users can interact with a virtual character by performing various activities such as eating, sleeping, playing, and taking medicine. The game is built with HTML, CSS, and JavaScript, fully responsive, and runs directly in the browser without any additional setup.

## 🎮 Key Features

- ✅ **Character Selection** (3 options available)  
- ✅ **Enter Character Name**  
- ✅ **Character reacts to activity** (changes appearance for 5 seconds and returns)  
- ✅ **Progress bar decreases** over time (idle or playing)  
- ✅ **Clock System**: Character levels up every 2 in-game hours  
- ✅ **Activity-based Progress Bar**: Increases when performing actions  
- ✅ **Level System**: 3 stages (baby → child → adult)  
- ✅ **Character evolves** visually as levels increase  
- ✅ **Dynamic Greetings** based on real-world time (morning, afternoon, evening)  
- ✅ **Background changes** with the time of day  
- ✅ **Mini Game** included (with different gameplay)  
- ✅ **Responsive design** across different screen sizes
